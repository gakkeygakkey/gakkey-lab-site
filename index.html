Bridge Meter & Bridge Plugin

常時表示の
メーターを、
小さく配置。

DAW内のBridge PluginやMacのオーディオ入力に加え、macOS 14.2以降ではSystem Audio Outputも直接キャプチャ。 入力に「System Audio Output」を選ぶだけで、Macから出ている音をそのままメータリングできます。

Flexible panels

必要なパネルだけ、
好きな場所へ。

ラウドネス、リサージュ、スペクトラムは一時的に分離表示できます。 Reference Levelも用途に合わせて変更できます。

Bridge Plugin System Audio Input System Audio Output AAX / VST3 / AU

Bridge Pluginからメーターへ。

AAX / VST3 / AU対応のBridge PluginをDAWに挿して、Bridge Meterへ表示用信号を送信。 音声そのものは変更しません。

Relative Time Counter

好きなところで、
カウンターを
0に！

MTC / MMCを使って、任意の位置をREL=0として表示。 クライアントから伝えられた「1:30」の場所へ、迷わず戻れます。

  • REL / ABS表示切り替え
  • MTC / MMC対応で、さまざまなDAWと連携
  • 相対時間またはタイムコードでロケート
  • Pro ToolsではPTSL SDKによる連携にも対応
  • Auto Talkback機能

動作環境 macOS 12 Monterey以降 / Universal Binary（Apple Silicon・Intel）
